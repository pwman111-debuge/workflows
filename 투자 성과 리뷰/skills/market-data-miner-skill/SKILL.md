# Market Data Miner Skill

이 스킬은 국내외 금융 포털에서 실시간 주가, 고저가 이력, 수급 데이터 등을 빠르고 정확하게 수집하고 정제하는 데이터 마이닝 전담 도구입니다.

## 🎯 Primary Purpose
분석에 필요한 시장 데이터를 누락 없이 확보하며, 특정 사이트 차단이나 오류 발생 시 자동으로 대체 수치를 확보하여 분석의 연속성을 보장합니다.

## 🚦 Data Source Priorities (우선순위)
분석 시 다음 순서대로 데이터를 조회합니다:

1. **Priority 1: 네이버 증권 (Naver Finance)**
   - 가장 상세한 종목 정보 및 투자자별 매매동향(수급) 제공.
   - 일별 시세 보기를 통해 과거 7~90일간의 OHLC(시가/고가/저가/종가) 데이터를 확보합니다.

2. **Priority 2: 토스증권 (Toss Securities)**
   - 네이버 증권 접근 불가 시 즉시 사용.
   - 직관적인 실시간 가격 및 변동성 지표 수집에 최적화.

## 🛠 Capabilities

### 1. 실시간 가격 및 고저가 스캔 (Price Scanning)
- 현재가뿐만 아니라, 특정 기간(7, 30, 90일) 내의 **최고가**와 **최저가**를 스캔합니다. (수익률 판정의 근거 데이터)

### 2. 수급 데이터 마이닝 (Supply/Demand Mining)
- 최근 5~10거래일간의 **외국인 및 기관의 순매수/매도 합계**를 산출합니다.
- 개인 투자자의 이탈 여부 및 메이이저 수급의 '매집' 신호를 판독합니다.

### 3. 장애 처리 및 대체 (Failover Logic)
- 텍스트 크롤링이 차단될 경우, 즉시 해당 종목의 **차트 캡처본을 분석(Vision)**하여 수치를 역산해 냅니다.
- 환율(USD/KRW) 등 거시 정보가 필요한 경우에도 동시에 수집합니다.

## 📋 Data Output Format (정제 규격)
모든 수집 데이터는 분석 워크플로우에 다음 형식을 참고하여 보고합니다:

| 항목 | 데이터 |
| :--- | :--- |
| **현재가** | [숫자]원 / [변동률]% |
| **이력(7/30/90d)** | 고점: [가격] / 저점: [가격] |
| **수급** | 외인: [매수/매도] / 기관: [매수/매도] |
| **출처** | [네이버/토스/비전분석] |

## 🚦 Usage Rules
- 황원장님의 우선순위 규칙(네이버 > 토스)을 엄격히 준수합니다.
- 숫자는 콤마(,)와 단위를 포함하여 가독성 있게 정제합니다.
- 데이터 수집 시점의 타임스탬프를 반드시 기록합니다.
