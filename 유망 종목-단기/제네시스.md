# 🚀 AntiGravity 단기 유망 종목 발굴 워크플로우 (제네시스)

이 워크플로우는 사용자가 **"제네시스 하자"**고 요청할 때 시작되며, 최고 수준의 금융 데이터를 바탕으로 최적의 단기 매매 종목 3개를 발굴합니다.

## 🛠️ 데이터 수집 및 분석 가이드라인
1.  **필수 데이터 소스**: 모든 시장 및 종목 검색 시 **토스증권(Toss Securities)**과 **네이버 증권(Naver Finance)** 사이트를 최우선적으로 이용합니다.
2.  **보조 데이터 소스**: 필요 시 인베스팅닷컴, 에프앤가이드 등 기타 검증된 금융 전문 사이트의 자료를 참조하여 교차 검증을 수행할 수 있습니다.
3.  **캡처 분석 보안 원칙**: 사이트의 보안 정책이나 로봇 배제 표준(robots.txt) 등으로 인해 데이터를 직접 텍스트로 읽어올 수 없는 경우, 반드시 **화면을 캡처(Screen Capture)한 후 인공지능 비전 분석**을 통해 수치와 차트 패턴을 판독하여 분석 리포트의 신뢰성을 확보합니다.

---

## 📋 수행 단계 (Operation Steps)

### Step 1. 시장 환경 및 거시 지표 점검
- **도구**: 네이버 증권/토스증권 메인
- **체크리스트**:
    - [ ] 코스피/코스닥 지수의 당일 방향성 및 거래대금 에너지
    - [ ] 외국인 및 기관의 전체 시장 수급 현합
    - [ ] 환율(USD/KRW) 추이 및 삼성전자 등 지수 영향력이 큰 종목들의 당일 수급

### Step 2. 수급 및 기술적 타점 스크리닝
- **도구**: 토스증권 '구매 상위/거래 급증', 네이버 증권 '거래상위/기관외인매매'
- **조건**:
    - [ ] 외국인과 기관의 **동반 순매수**가 포착된 종목
    - [ ] 당일 거래량이 전일 대비 1.5배 이상 폭발적으로 급증한 종목
    - [ ] **차트 분석**: 이동평균선 정배열 초기 국면 또는 골든크로스 확인 (필요 시 차트 캡처 분석)
    - [ ] **보조지표**: RSI(50~70) 및 MACD 시그널의 양의 모멘텀 점검

### Step 3. 리스크 필터링 및 공시 검증
- **도구**: 네이버 증권 뉴스/공시 탭
- **조건**:
    - [ ] 최근 5거래일 이내 종목에 부정적 영향을 줄 수 있는 **악재 공시** 여부 확인
    - [ ] 현재 시장의 주도 테마(섹터) 내 대장주 지위 여부 확인

### Step 4. 최종 리포트 생성 및 배포
- [ ] **보고서 작성**: 지정된 출력 양식(시장 환경 요약 + 종목별 상세 전략) 준수
    - **[MDX 필수 필드(스키마) 준수]**:
        *   `title`: 리포트 제목
        *   `ticker`: "GENESIS" (종합 리포트 시 고정)
        *   `market`: "KOSPI" 또는 "KOSDAQ"
        *   `term`: "short"
        *   `date`: "YYYY-MM-DD"
        *   `currentPrice`, `targetPrice`, `stopLoss`: 0 (종합 리포트 시 고정)
        *   `status`: "active"
        *   `summary`: 본문 핵심 요약
        *   `tags`: 키워드 리스트
- [ ] **파일명**: `YYYYMMDD_단기유망종목_발굴리포트.md`
- [ ] **사용자 확인**: 리포트 작성 후, **GitHub 푸시 여부를 사용자에게 반드시 확인**하고 승인 시 진행합니다.

## 🚦 운영 원칙
1.  **데이터 무결성**: 분석 시점의 실시간 수치를 소수점까지 정확하게 기록합니다.
2.  **명확한 가이드라인**: 모든 추천 종목에는 현재가 대비 현실적인 **목표가**와 리스크 관리를 위한 **손절가**를 반드시 포함합니다.
3.  **시각적 판독**: 데이터 추출 차단 시 주저 없이 캡처 분석을 수행하여 공백 없는 리서치를 수행합니다.
4.  **손절가 필히 명시**: 종목별로 리스크 관리를 위한 명확한 목표가와 손절가를 포함합니다. (위 2번 항목과 통합 관리)
