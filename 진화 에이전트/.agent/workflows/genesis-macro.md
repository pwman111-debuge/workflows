# 🚀 제네시스-매크로 (Commander)

이 워크플로우는 시장의 '온도'를 체크하여 전체 투자 전략의 가이드라인과 팀 소집 범위를 결정하는 **사령관**입니다.
사용자가 **"제네시스"**라고 명령하면 아래의 시황 분석 프로세스를 시작합니다.

---

## 0. 날짜 및 요일 인지 (Master Protocol)

- **날짜 확인**: 현재 시스템 날짜를 확인하여 **금요일** 혹은 **매월 첫째 주 월요일**인지 판별합니다.
- **분객 처리:**
  - **평일(월~목)**: [매크로 분석] → [단기 종목 발굴(Executor)] + **[성과 복기(Reviewer)]**
  - **매주 금요일**: [매크로 분석] → [단기 종목 발굴(Executor)] + [중기 종목 발굴] + **[성과 복기(Reviewer)]**
  - **매월 첫째 월요일**: [매크로 분석] → [단기 종목 발굴] + [장기 종목 발굴] + **[성과 복기(Reviewer)]**

> **[성과 복기(Reviewer) 기준 시점]**
> - 단기 종목: 분석 시점 기준 **약 7일 전** 추천 종목
> - 중기 종목: 분석 시점 기준 **약 30일 전** 추천 종목
> - 장기 종목: 분석 시점 기준 **약 90일 전** 추천 종목

- **보고**: 황원장님에게 "오늘은 [요일]이므로 [단기/중기/장기] 분석 팀을 가동합니다"라고 먼저 보고합니다.

---

## 0.5. 성과 복기 데이터 및 방법론 확인 (Review Consumption)

- **복기 대상 기간:**
  - **단기 종목 복기**: 분석 시점 기준 약 **7일 전** 추천 종목의 수익률 및 논리 검증
  - **중기 종목 복기**: 분석 시점 기준 약 **30일 전** 추천 종목의 수익률 및 논리 검증
  - **장기 종목 복기**: 분석 시점 기준 약 **90일 전** 추천 종목의 수익률 및 논리 검증
- **피드백 데이터 조회**: `.agent/memory/state.json`의 `compounding_engine` 데이터를 분석합니다.
- **전략 및 방법론 반영:**
  - **승률 저하**: 최근 승률이 40% 미만일 경우, 전체 지수를 한 단계 다운그레이드하여 보수적으로 대응합니다.
  - **방법론 결함 발견**: 복기에서 지적된 **'매크로 판정 로직의 오류'**(예: 특정 경제 지수 무시 등)가 있다면, 오늘의 판정 루틴에 해당 변수를 강제로 추가합니다.
  - **경고 패턴**: 현재 장세가 `warning_patterns`와 일치할 경우 방어 중심 대응을 명시합니다.

## 1. 데이터 수집 단계 (Data Sourcing)

AI 에이전트는 다음 사이트를 검색하여 **실시간 데이터**를 수집합니다:

- **네이버 증권 (finance.naver.com)**
  - 코스피/코스닥 지수 현황
  - 외국인·기관 수급 상황 (순매수/순매도 금액)
  - 업종별 등락률 및 주도 업종 파악
  - 주요 뉴스 헤드라인 스캔
- **토스 증권 (tossinvest.com)**
  - 실시간 테마별 수익률 순위
  - 투자자 관심 종목 Top 10
  - 급등 테마 및 이유 분석
- **구글 파이낸스 (finance.google.com)**
  - 원/달러 환율 및 방향성
  - 미 증시(나스닥/S&P500/다우) 마감 지수
  - 글로벌 시장과의 연결성 분석
- **기타 검증 사이트**
  - **인베스팅닷컴**: 원자재(WTI, 금), 글로벌 금리 동향
  - **CNN Fear & Greed Index**: 시장 심리 온도계 수치 확인

> ※ 필요시 **브라우저 캡처 및 OCR**을 활용한 이미지 분석을 병행합니다.

## 2. 3단계 레이어 심층 분석 (3-Layer Deep Analysis)

수집된 데이터를 바탕으로 **입체적 분석**을 수행합니다:

### [Layer 1] 거시 분석 (Macro)
- 금리: 미국 국채 10년물 금리 방향 및 한국은행 기준금리 스탠스
- **환율 (Critical Threshold)**:
  - 원/달러 1,300원 기준선 상하 여부 및 방향성 체크.
  - **🚨 위기/기회 특수 시나리오**: 환율 **1,500원** 돌파(또는 임박) 시, `strategic_note`에 '환율 수혜 수출주' 및 '고유가 수혜 에너지주' 가중치 2.5배 상향 지시를 반드시 포함합니다.
- 글로벌 거시 지표: 미국 CPI, PMI, 고용 데이터 등 주요 경제 지표
- 시장 에너지 판정: Bull / Bear / Neutral 최종 판정

### [Layer 2] 중기 분석 (Mid-term)
- **지수 기술적 위치**: 코스피/코스닥의 주요 이동평균선(20일·60일·120일) 위치
- **외국인 수급 지속성**: 최근 5~20일 외국인 순매수 흐름의 방향성
- **섹터 로테이션 패턴**: 주도 섹터 이동 경로 및 차기 주도 섹터 예측

### [Layer 3] 단기 분석 (Short-term)
- **당일 실시간 수급 쏠림**: 개인·기관·외인의 매수 집중 업종 파악
- **주도 테마**: 당일 급등 테마 및 모멘텀 지속 가능성 판단
- **변동성 요인**: 돌발 뉴스, 공매도 이슈, 프로그램 매매 동향

---

## 🤝 3. [소통] 전략 가이드라인 선언 (Blackboard Write)

분석 완료 후, `.agent/memory/state.json`의 `macro_analysis` 필드를 업데이트합니다.

- **`market_energy`**: 분석된 등급 (Bull / Bear / Neutral) 기록.
- **`strategic_note`**: 종목 발굴팀(Executor / Analyst / Visionary)에게 보낼 **'사령관 지시 사항'** 작성.
  - (예: "장기 비전은 맑음, 중기 실적 확인 필, 단기 수급 보수적")

## 4. 시황 분석 보고서 생성 (MDX Format)

분석 결과는 **MDX 파일** 형식으로 작성됩니다.
- **파일명**: `YYYY-MM-DD-market-analysis-genesis.mdx`

### 필수 Frontmatter 항목

| 항목 | 내용 |
|------|------|
| `title` | 보고서 제목 |
| `date` | 작성 날짜 (YYYY-MM-DD) |
| `category` | `Genesis` **(반드시 이 값 사용)** |
| `ticker` | 종목 코드 (예: `"005930"` 입력 시 실시간 차트 자동 생성) |
| `summary` | 콘텐츠 요약 (1~2문장, 배포 필수 항목) |
| `tags` | 관련 키워드 리스트 |

### 필수 포함 본문 섹션

- **[확인 내용]**: 오늘 수집한 데이터 중 핵심 시황 사실 정리
- **[리스크 판정]**: 하락/변동성 확대 가능 요인 명시 (Bull이라도 반드시 작성)
- **[오늘의 매매 전략]**: 당일 대응 전략 (공격적/중립/방어적 중 선택 후 근거 제시)
- **[종합 투자 액션 플랜 (Do & Don't)]**:
  - **Do**: 오늘 적극 관심 가져야 할 섹터·종목 방향성
  - **Don't**: 오늘 반드시 피해야 할 포지션·섹터·행동

## 5. GitHub 연동 및 배포 (Deploy)

> [!IMPORTANT]
> **GitHub push 여부는 반드시 황원장님이 직접 결정합니다.**
> 에이전트는 보고서 생성 완료 후 푸시 승인을 요청하며, **황원장님의 명시적 승인 없이는 절대 배포하지 않습니다.**

- 보고서 생성 완료 → 에이전트가 푸시 승인 요청
- **황원장님 승인 시** → GitHub 저장소로 push 실행
- **황원장님 보류 시** → 로컬에만 저장, 추후 재요청 대기

---

## 6. 자가 진화 (Self-Evolution)

- **성능 회고**: 데이터 수집 지연 여부 및 황원장님의 피드백을 반영하여 로그를 남깁니다.
- **업그레이드 루프**: 로그 누적 시 안팀장이 직접 워크플로우를 최적화합니다.
- **경고 패턴 축적**: 시장 급변 시나리오를 `warning_patterns`에 추가하여 다음 분석에 반영합니다.
